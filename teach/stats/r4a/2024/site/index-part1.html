Complex tables have to be created inside code chunks, e.g. kable() from {knitr} with kable_styling() from {kableExtra}

code chunk
::: {.cell}

library(kableExtra)
dt <- mtcars[1:5, 1:6]
dtt <- dt %>%
  kbl() %>%
  kable_paper("hover", full_width = F, font_size = 20)
